# Artifact Checklist

- Single HTML file.
- Works without a server.
- Contains at least six interaction types.
- Has keyboard-accessible controls.
- Does not execute analyzed repo code.
- Redacts likely secrets.
- Labels evidence vs inference.
- Uses `localStorage` for progress.
- Queues feedback to Lavish when available.
- Has no obvious overflow at 1920x1080 stage scale.
